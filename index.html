Vulns - Open Source Scrapper Project - National Vulnerability Database

Work in progress https://github.com/earthroot/nvd-scrapper

This POC ilustrates how to centralize CVE information and use a serverless arhitecture to display relevant dashboards

  • Launch a container (https://hub.docker.com/r/clglavan/nvd-scrapper) in a serverless environment like github actions and send data to a gcp bucket
  • Pull data from the gcp bucket into a BigQuery dataset
  • Create a datastudio report with BigQuery dataset as data source
