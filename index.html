This POC ilustrates how to centralize CVE information and use a serverless arhitecture to display relevant dashboards

  • Launch a container (https://hub.docker.com/r/clglavan/nvd-scrapper) and send data to a gcp bucket
  • Pull data from the gcp bucket into a BigQuery dataset (https://hub.docker.com/r/clglavan/bigquery-job)
  • Create a datastudio report with BigQuery dataset as data source
  • Automate everything in a serverless environment like github actions or gitlab ci/cd
  • Check out this docker-compose example
